# 票务平台基础规则

更新时间: 2026-03-27

## 实名制

- 演唱会类活动通常采用实名制购票和实名制入场。
- 下单前需要填写真实观演人信息，姓名与证件号必须和入场证件一致。
- 一张票通常对应一位实名观演人，填写后是否允许修改以下单页说明为准。

## 退票说明

- 不同活动的退票规则可能不同，最终以活动详情页展示为准。
- 如果活动详情页未单独说明，平台默认不承诺无条件退票。
- 临近演出时间的订单通常会限制退票或收取手续费。

## 入场须知

- 电子票订单一般通过二维码或动态码验票入场。
- 观演人需携带与实名信息一致的有效证件。
- 场馆通常会限制危险物品、专业摄录设备和大件行李入场。
